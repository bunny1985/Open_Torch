Open_Torch by bunny1985

Prosta latarka dal samsunga wave w c++ Bada Api

[pl] Lataraka dla systemu bada wykorzystująca diodę LED. Najprostrza z możliwych. żadnych dodatkowych bajerów. Zasada działania: Latarka nagrywa film. Włącza kamerę i tworzy plik niskiej jakości, po czym przy wyłączaniu anuluje nagrywanie. Wykorzystuje przy tym największą dostepną pamięć ( tel/karta) [en] It is the simplest flash light application for samsung wavee bada os. How it works: It's creating low quality movie on your device and when you turn this app of the movie is canceled. It uses as much memory as it can to not blinking.

Dependencies

BADA API 1.0and higher

Install

Produkt , gdy osiągnie dojrzałość, powinien być dostępny w samsungapps . Ponieważ kod żródłowy jest dostępny istnieje także możliwość instalacji za pomocą bada IDE ( po zimportowaniu i zbudowaniu )
